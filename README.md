A raycaster written in Rust using SDL2 graphics library

![image](example.png)

## Requirements
The packages below
```
base-devel
sdl2
sdl2_image 
```

## Compiling and Running the Examples

Once you have the packages below installed along with rust, you can build the raycaster using
```
cargo build
```
